## 개요

본 체크리스트는 TheK-NEXT 시스템의 기존 페이지에 JP i18n 정책을 적용할 때 사용합니다.  
모든 항목을 확인하고 체크 완료 후 PR을 생성하세요.

---

## 1. 공통 헤더/메뉴 로더 (Common Header/Menu Loader)

### 1.1 번역 키 적용

- [ ] 헤더 타이틀이 번역 키로 교체됨
- [ ] 메뉴 항목이 모두 번역 키로 교체됨
- [ ] 서브메뉴 항목이 모두 번역 키로 교체됨
- [ ] 로그아웃/설정 등 공통 버튼이 번역 키로 교체됨
- [ ] 사용자 이름 표시 영역에 플레이스홀더 적용됨

### 1.2 혼용 검출 체크

- [ ] 한글 문자(`[가-힣]`) 포함 여부 확인 → 0건
- [ ] 한국어 조사(`은/는/이/가/을/를/의`) 포함 여부 확인 → 0건
- [ ] 한국어 어미(`습니다/합니다/입니다`) 포함 여부 확인 → 0건
- [ ] 하드코딩된 일본어 텍스트 없음 확인

### 1.3 UI 검증

- [ ] 일본어 표시 시 텍스트 잘림 없음
- [ ] 메뉴 레이아웃 깨짐 없음
- [ ] 일본어 폰트 정상 렌더링

## 5. 공통 검증 항목

### 5.1 날짜/시간 형식

- [ ] 날짜 표시: `yyyy年MM月dd日` 형식 적용
- [ ] 시간 표시: `HH:mm` 형식 적용
- [ ] 날짜+시간: `yyyy年MM月dd日 HH:mm` 형식 적용

### 5.2 통화 형식

- [ ] 통화 표시: `¥1,000` 또는 `1,000円` 형식 적용
- [ ] 천 단위 구분자 적용 확인

### 5.3 경어 수준

- [ ] 사용자 대상 메시지: 정중어(〜です、〜ます) 사용
- [ ] 에러 메시지: 정중어(〜してください) 사용
- [ ] 반말(タメ口) 사용 없음 확인

### 5.4 로케일 전환 테스트

- [ ] ja_JP → ko_KR 전환 시 정상 표시
- [ ] ko_KR → ja_JP 전환 시 정상 표시
- [ ] 브라우저 언어 자동 감지 정상

## 체크리스트 사용 방법

1. 적용할 페이지 유형에 해당하는 섹션 확인
2. 각 항목을 순서대로 체크
3. 혼용 검출 체크는 반드시 자동화 도구 사용
4. 모든 항목 완료 후 PR 생성
5. 백설이(오케스터)에게 검수 요청
